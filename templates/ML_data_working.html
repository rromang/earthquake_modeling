Earthquake Prediction

Regression Model

Extra Trees Regressor Model

This ensemble model was used for training a model to predict features from the USGS Earthquake data collected.

The features used for training the model were:

  • Magnitude
  • Latitude
  • Longitude
  • Depth
  • Time (converted to Unix)

The model was run 9 times with alternating predictors and target features.

  • Magnitude Prediction:

    • Magnitude (y)
    • Latitude (x)
    • Longitude (x)
    • Depth (x)
    • Unix Time (x)
  • Magnitude Prediction (No Unix Time):
    • Magnitude (y)
    • Latitude (x)
    • Longitude (x)
    • Depth (x)
  • Latitude Prediction:

    • Magnitude (x)
    • Latitude (y)
    • Longitude (x)
    • Depth (x)
    • Unix Time (x)
  • Latitude Prediction (No Unix Time):
    • Magnitude (x)
    • Latitude (y)
    • Longitude (x)
    • Depth (x)
  • Longitude Prediction:
    • Magnitude (x)
    • Latitude (x)
    • Longitude (y)
    • Depth (x)
    • Unix Time (x)
  • Longitude Prediction (No Unix Time):
    • Magnitude (x)
    • Latitude (y)
    • Longitude (x)
    • Depth (x)
  • Depth Prediction:
    • Magnitude (x)
    • Latitude (x)
    • Longitude (x)
    • Depth (y)
    • Unix Time (x)
  • Depth Prediction (No Unix Time):
    • Magnitude (x)
    • Latitude (x)
    • Longitude (x)
    • Depth (y)
  • Unix Time Prediction:
    • Magnitude (x)
    • Latitude (x)
    • Longitude (x)
    • Depth (x)
    • Unix Time (y)

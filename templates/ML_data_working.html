Extra Trees Regressor Model

This ensemble model was used for training a model to predict features from the USGS Earthquake data collected.

The features used for training the model were:

  • Magnitude
  • Latitude
  • Longitude
  • Depth
  • Time (converted to Unix)

The model was run 9 times with alternating predictors and target features.

Latest Work

The current models were run with 50,000+ rows. The test dataset portion is 33% of the rows from the total dataset. The number of estimators was a max of 80. The code will use the number of estimators with the best score and use that model to predict the target.

Latitude model with all the features had the best performance, with the model explaining ~90% of the variability from the predictor features.

The next best model in terms of performance was depth without time as a predictor. This model was ~86% score.

As expected, the model to predict time perform dismally, with essentially no variability explained by it.
